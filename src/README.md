# Hash Tables Implementation in Java

This repository contains 10 real-world system design problems implemented using Hash Tables in Java.

## Problems Implemented
1. Username Availability Checker
2. Flash Sale Inventory Manager
3. DNS Cache with TTL
4. Plagiarism Detection System
5. Real-Time Analytics Dashboard
6. Distributed Rate Limiter
7. Autocomplete System
8. Parking Lot (Open Addressing)
9. Two-Sum Financial Transactions
10. Multi-Level Cache System

Each problem is committed separately to demonstrate incremental development.
